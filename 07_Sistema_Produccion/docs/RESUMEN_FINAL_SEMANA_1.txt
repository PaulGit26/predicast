🔧 API ENDPOINTS DISPONIBLES
═══════════════════════════════════════════════════════════════════════

POST /api/v1/auth/register
  → Registrar nueva empresa
  → Response: token + tenant_id

POST /api/v1/auth/login
  → Login de usuario
  → Response: token

POST /api/v1/predict
  → Hacer predicción individual
  → Response: prediction + confidence

POST /api/v1/predict/batch
  → Predicciones en batch (múltiples)
  → Response: lista de predicciones

GET /api/v1/model/info
  → Info del modelo (features, R², etc)
  → Response: metadata del modelo

GET /api/v1/predictions
  → Historial de predicciones del tenant
  → Response: lista con timestamp
